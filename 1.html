O que é Javascript?

Qual foi a motivação para criar o Javascript?

​A motivação para a criação do JavaScript surgiu no contexto da necessidade de tornar as páginas web mais dinâmicas e interativas. Nos anos 90, o HTML era estático e não oferecia recursos para criar experiências ricas para os usuários. A Netscape, uma das pioneiras na navegação web, percebeu que era necessário adicionar elementos dinâmicos às páginas, permitindo interações como validação de formulários e manipulação de conteúdo sem a necessidade de recarregar a página inteira. Para atender a essa demanda, em 1995, a empresa contratou Brendan Eich para desenvolver uma linguagem de script que possibilitasse essas funcionalidades. Assim nasceu o JavaScript, inicialmente chamado de LiveScript, com o objetivo de complementar o HTML e proporcionar uma experiência mais interativa aos usuários na web.

Qual a diferença entre execução no navegador e no servidor?

A execução de código no navegador e no servidor difere principalmente no local e propósito da execução. No navegador, o código é executado diretamente na máquina do usuário, geralmente com JavaScript, e tem como objetivo interatividade e manipulação da interface do usuário. Isso permite uma experiência rápida e dinâmica, mas o código pode ser acessado e modificado pelo usuário, o que pode representar um risco de segurança.

Qual é o conceito de sandbox no navegador?

O conceito de sandbox no navegador refere-se a uma técnica de segurança que isola a execução de scripts e conteúdos potencialmente perigosos dentro de um ambiente restrito e controlado, para evitar que esses conteúdos afetem negativamente o sistema do usuário ou acessem recursos não autorizados.
